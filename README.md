# BharatSwasth AI

BharatSwasth AI is a GenAI-powered rural healthcare platform for India focused on
voice-first interaction, Indian language support, offline-first experience,
and rapid triage guidance for citizens and frontline workers.

## Problem
Rural regions face shortages of doctors, language barriers, and delayed diagnosis.

## Solution
Our AI assistant provides multilingual voice input, early risk detection,
clinic discovery, reminders, and government scheme awareness.

## Built With
- Google Cloud Vertex AI (Gemini models)
- Cloud Run
- Firestore
- BigQuery
- Speech-to-Text APIs

## Status
Concept + Architecture stage (submission for AI for Bharat challenge).
